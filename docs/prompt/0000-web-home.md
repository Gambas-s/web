# 웹 화면 개발

## 공통 요구사항
- 모바일 화면과 별도 컴포넌트로 분리
- 현재 서버가 없으므로 mock 데이터 도입
- 필요한 이미지 @apps/web/public에서만 사용
- 사이드바와 그 안에 있는 쓰레기통은 공통 컴포넌트로 관리
